Pós Instalação

Ok, você instalou o seu Cajado e iniciou a máquina e irá cair na tela de login... O primeiro passo é claro logar como root para começar a brincadeira.

Criando conta de usuário

Se você quer ter um ambiente convencional para usar no dia a dia você ira necessitar criar uma conta de usuário comum. Não é seguro utilizar o root para tarefas básicas do dia dia! A não se que você realmente saiba o que está fazendo.

Para criar uma conta utilizaremos o comando adduser. Ele é um comando interativo e te fará um monte de perguntas, bastanto responder. 

adduser NomeDoUsuário

Ajustando Teclado para ABNT2 e Linguagem do Sistema para PT-BR

Se você utiliza o padrão ABNT2 e gosta do sistema em Português do Brasil siga este passo a passo, se não pode pular.

Existem duas categorias de reconhecimento da linguagem do teclado.
A primeira forma consiste em trocar o padrão via TTY, provavelmente você está neste ambiente agora.
A segunda forma é trocar o valor do teclado para interface gráfica. Recomendo trocar as duas :)

Para a primeira forma via TTY, vamos criar um serviço de inicialização em /etc/rc.d/rc.keyboard. Você pode abrir seu editor de texto favorito OU fazer via linha de comando. Para nosso mêtodo utilizaremos a linha de comando com o comando cat fazendo um heredocument. 

cat >> /etc/rc./rc.keyboard << EOF
#!/bin/sh
# Teclado para ABNT2

if [ -x "/usr/bin/loadkeys" ]; then
    /usr/bin/loadkeys br-abnt2.map
fi

EOF

Feito isso dê permissão de execução para inicialização do serviço no boot do Cajado! 

chmod +x /etc/rc.d/rc.keyboard

O primeiro passo está feito, porém quando você inicializar a interface gráfica o teclado não vai se ajustar para o ABNT2. O que fizemos foi para o TTY se ajustar. Vamos agora ajustar para interface gráfica.

Existe um arquivo chamado 90-keyboard-layout.conf que está localizado em /usr/share/X11/xorg.conf.d/. Vamos fazer uma cópia deste arquivo para o diretório /etc/X11/xorg.conf.d/ e logo em seguida abrimos ele com o nosso editor favorito. 

# Copiando Arquivo

cp /usr/share/X11/xorg.conf.d/90-keyboard-layout.conf \
/etc/X11/xorg.conf.d/

# Abrir Arquivo
vi /etc/X11/xorg.conf.d/90-keyboard-layout.conf

Ao abrir o arquivo procure a linha: Option "XkbLayout" "us" e substitua "us" por "br" ficando: 

Option "XkbLayout" "br"
